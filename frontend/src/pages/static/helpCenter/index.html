用车安全

检查车辆安全四步走

  • 第一步：外观 查看是否有明显划痕，如有，立即拍照上传，作为您租车的免责声明。
  • 第二步：轮胎 查看轮胎是否亏气或存在明显裂痕。
  • 第三步：仪表盘和灯光 查看仪表盘是否有红灯亮起，如有红灯亮起，请停止用车并及时上报客服；检查各车灯系统，如拨转左右转向灯，出现频闪时，则为转向灯失灵；为了避免用车过程中电量不足，请先查看车辆实际续航里程，续航里程大于计划用车里程的20公里方可用车。
  • 第四步：车辆启动后，缓踩刹车，查看制动系统是否正常。
  • 说明：“立刻出行”在车辆上线时，都会进行车辆安全检查，但为了保证您的用车安全，用车前请进行安全检查。如出现2、3、4问题时，请及时通知客服，建议您选择更换车辆或使用其他方式出行。

预定

  • • 租车如何收费？

    1、用户在分时用车期间发生的费用:
    基础收费方式：里程费用+时间费用。
    标准收费：2元/公里+0.2元/分钟（不同城市和车型有不同的限时优惠）。
    以广州为例：限时优惠费用为1.0元/公里+0.1元/分钟，订单行驶10公里+60分钟，应付16元。

    2、动态加减价说明：

    a.根据需求及您还车地点车辆调配原因，部分车辆会进行动态价格调配。

    b.您的实际还车地点与计划选定的还车地点不同，在还车时会执行当时的价格调控策略。

    3、临停费

    在分时用车期间在“立刻出行”签约停车网点产生的停车费用，此类停车场不会向用户直接收取停车费用。会由“立刻出行”代缴。

  • • 注册登录后为什么不能用车？

    为了保障您的合法用车权益，在注册后，需要完成身份认证，需在APP内提交本人的身份证和驾驶证信息，经审核通过后即可下单使用车辆。

  • • 用户注册资格是什么？

    用户必须符合“立刻出行”标准驾驶员的资格要求。在提交申请时，用户应具备：

    （A）符合法律规定的最低年龄要求；

    （B）驾照有效并能够准驾自动挡车型，没有被吊销、没 收、失效或以其它方式作废。

    如用户的驾照在任何时间被吊销、没收、失效或其状态发生变化，用户有义务立即告知“立刻出行”, 用户将不被批准使用或预定任何“立刻出行”车辆。

    “立刻出行”用户可通过手机端应用市场下载“立刻出行”客户端后，进入用户注册页面，按要求填写注册信息，上传身份证及驾照实际照片，完成用户注册过程。“立刻出行”用户资料审核通过后（资料提交成功15分钟之内），即可使用“立刻出行”的分时租车服务。

  • • 身份认证审核为什么不通过？

    1）提交的身份证和驾驶证照片必须为本人真实的证件。

    2）提交的证件照片必须按要求上传，且完整无残缺，客服人员能识别清楚。

    3）本人手持身份证照片必须与上传的证件照信息相匹配。

    4）相关信息不符合国家相关公共管理部门认证要求

    5）审核不通过，需要您依照以上标准重新提交身份认证。

  • • 登录时为什么收不到短信验证码？

    1）请先确认手机号是否输入正确。

    2）检查手机通讯信号是否存在异常，如发现信号异常，可重新换一地点使用，或重启手机后再使用。

    3）安卓手机请检查一下是否安装了短信拦截软件，如有安装请关闭拦截功能重新获取即可。

  • • 打开APP地图首页为什么定位失败？

    请检查手机是否设置了拒绝APP获取位置信息的权限，如果是拒绝获取位置信息，请更改为允许即可。

  • • 怎么领用和使用优惠券？

    如果账户中有优惠券，在订单结算时，系统将自动匹配可使用的优惠券，也可根据用车情况选择使用相应优惠券。

  • • 可以跨城市用车吗？

    不可以，跨城市无法操作还车。由于行程较远，如行程中出现没电、趴窝等情况所产生的费用需要用户自行承担。

  • • 名词解释

    授权驾驶人
    指“立刻出行”用户本人，年满18周岁以上，具有有效驾驶证。

    付费
    指用户支付的所有款项，包括租金、里程费，时长费以及服务条款中规定的其他相关费用。

    用户
    指申请并通过“立刻出行”审核的有效用户。

    立刻出行
    指当地的“立刻出行”运营机构“天津山和朋友们科技有限公司”及下属公司，以及天津以外的分支机构。

    共享使用期限
    指从用户确定用车时间到用户将汽车归还至指定汽车还车地点、并按“立刻出行”要求将车停放妥当，进行充电并确认还车的期间。

    预定时间
    指用户与“立刻出行”预先约定的取车时间。

    车辆
    指用户预定的“立刻出行”租赁车辆“立刻出行”可能提供的任何替换车辆。

    车辆状态
    指“立刻出行”车辆外观、市场价格、组件、配件、系统情况，汽车内外清洁程度等与车辆工作状态相关的情况。

用车

  • • 找不到取（还）车网点怎么办？

    部分取（还）车网点可以通过点击网点全景地图方式自行查看，如果仍然找不到网点您可以致电”立刻出行”400客服，咨询网点详细位置即可。

  • • 用车行程中出现故障怎么办？

    用车行程中如出现故障请不要慌乱，请第一时间将车辆停至不妨碍其他车辆通行的地方并致电400客服电话，我们会协助您联系道路救援。

  • • 用车行程中发生了交通事故怎么处理？

    在用车中发生事故请勿惊慌，请及时在车后摆放安全警告标志-三脚架，并在拍照取证后联系”立刻出行”客服，客服将协助您处理。

  • • 在取车网点找不到车怎么办？

    请您优先使用软件内“寻车”按钮通过声音和车灯闪烁寻找车辆，如仍无法找到车辆联系客服咨询前一单用户确认车辆停放位置或更换，变更出行方式。

  • • 车门打不开怎么办？

    车门打不开时，请确认软件提示，如果提示“离车太远”，请尝试重新开关手机的GPS定位功能，然后把软件重新启动再尝试开门，启动软件时如果提示“请求使用您当前的位置”请点击允许。若仍然无法打开车门，请联系”立刻出行”客服进行处理。

  • • 还车时没有停车位了怎么办？

    还车时如果遇到没有停车位的情况，请优先找到停车场管理员，协助停车，如果实在无法停进网点，那么可以更改还车网点选择附近的其他网点还车。如果以上操作都不方便进行时，请致电”立刻出行”客服进行处理。

  • • 还不了车怎么办？

    如果遇到无法还车的情况，请您检查是否已经把车辆开到指定的停车场，如果已经到达相应的车场仍无法还车，请您及时致电”立刻出行”客服：400-666-2333联系处理。

  • • 结单后支付不了怎么办？

    如果您无法使用余额结算订单，请您核实，是否为先操作了还车，后充值的余额，如果是这种情况，请您联系客服为您操作通过余额结算。

  • • 取车时应注意些什么？

    1）请仔细检查车辆外观、内饰是否存在已有的损坏，如发现损坏情况请及时拍照取证并上传，或致电客服：400-666-2333，以免给您带来不必要的损失。

    2）使用车辆前，请先确认车辆电量是否满足本次出行里程要求，若电量不足请联系”立刻出行”客服，并通过APP结束该订单，重新预定其它车辆。

    3）在出行前，请确认车辆仪表、车灯、刹车、车窗等有无异常，如发现异常请联系”立刻出行”客服，以免为您的出行带来不必要的麻烦。

  • • 还车时应注意些什么？

    1）还车时请将车辆应停放在”立刻出行”指定网点和停车位，若停放在非指定网点产生停车费用， “立刻出行”有权向您收取。

    2）停车后请关闭车窗、电动机，锁闭车门，否则无法结束订单。禁止将车钥匙携带离开。

    3）还车时您有义务保持车辆内部整洁，请将用车途中产生的垃圾收走，保证车辆内部卫生和取车时相同即可。

    4）在下车前请带走您的全部个人物品，请勿遗留在车上。

    5）还车后需要确认车辆是否完好无损，并对车辆相应位置拍照上传作为凭证，以免后期造成不必要的纠纷。

  • • 我租的车其他人可以开吗？

    不可以。您租的车只能由您本人驾驶，不允许借由他人驾驶。只有您才能取车，用车和还车。若其他人使用您预定的车，发生交通事故，保险公司将不予理赔，损失将由您全部承担。

  • • 能跨城市还车吗？

    不可以跨城市还车，暂未开通跨城市还车服务。

售后

  • • 有违章的话，谁去处理？

    租客在租车期间的违章情况，“立刻出行”将通过短信，app推送等方式通知用户。用户应在接到通知日起7天内自行前往工商银行或交管部门处理。确认该违章在用户的用车时间段内的情况下，如用户超过7天未处理，将被视为用户自动委托“立刻出行”处理相关违章。

    “立刻出行”有权在违章用户账户内扣除违章罚款等同金额及每次违法行为150元的代办费用（违章扣分按200元每分扣除）。以上收费金额为广州地区标准，其它地区按当地实际收费标准执行。

    对于吊销驾照或单次违章扣分6分以上的严重交通违法行为，“立刻出行”不接受委托处理。

  • • 在行程结算时对账单有异议怎么办？

    如在行程结算时，您对行程的费用产生异议，比如行程公里数和用车时间的数据不准确等，您可以致电客服电话：400-666-2333，客服人员将协助您完成进行数据核对后在进行支付。

  • • 租车可以开发票吗？

    可以，请到设置中选择“开发票”选项。按照客户端中的提示进行填写，我们会在5-10个工作日将发票寄到对应的邮寄地址。 （可开发票额度≥200元，即可免费为您邮寄）

  • • 车辆都有哪些保险？

    1、租客义务：

    在承租人租赁期间发生交通事故，承租人必须第一时间保留现场证据，并通知400-666-2333，进行报备。

    2、保险种类：

    交强险及商业险

    3、险种及限额：

    a、交强险险种及限额:死亡伤残11万；医疗费用1万；财产损失2千。

    b、商业险险种及最高限额（不同车辆可能涉及不同商业险种，详情咨询400-666-2333），包括不限于：车辆损失险8.48-13.3万；车上人员险1-2万/座；三者险5-50万；划痕险2千；涉水险3万；玻璃险等。不同车辆涉及不同险种，具体以车辆实际保险为准。

    4、保险投保公司：

    中国人保和太平洋保险等。不同车辆涉及不同保险公司，以车辆实际保险公司为准。

    5、以上最终解释权归立刻出行公司所有。

  • • 将个人物品落在车上怎么办？

    如果您不慎将自己的个人物品遗忘在车内，请您第一时间联系客服为您找回。但由于车辆属于共租运营车辆，不能确保一定可以找回，但我们一定会尽力为您寻找。建议您养成良好的用车习惯，在还车锁门之前，用1分钟的时间再次确认一下是否还有个人物品没有取出。

  • • 租车后我的个人隐私如何保障

    “立刻出行”的车辆装载有卫星定位系统（GPS）、行车记录仪以及远程处理系统。 “立刻出行”承诺保护每位用户的隐私。 用户同意“立刻出行”检索并使用车载远程处理装置所记录所有的资料。 “立刻出行”按隐私政策限制其员工及其它有关方使用用户的个人信息，防止非公开个人资料遭到非授权散布。
